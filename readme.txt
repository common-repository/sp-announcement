== Installation ==

1. Visit `Plugins > Add New`
2. Search for `WP Announcement`
3. Install WP Announcement once it appears.
4. Activate WP Announcement from your Plugins page.
5. Go to "after activation" below.

= Manually =

1. Upload the `sp-announcement` folder to the `/wp-content/plugins/` directory
2. Activate the WP Announcement plugin through the 'Plugins' menu in WordPress
3. Go to "after activation" below.

= After activation =

1. You should see the WP Announcement menu.
2. Go to that menu and create notice/announcement.
3. You're done!
